## OAuth client IDs

The Google provider requires OAuth 2.0 client IDs per platform (web,
iOS, Android). These are auto-generated by Firebase when Google sign-in
is enabled and live in:

- Firebase Console → Authentication → Sign-in method → Google
- Google Cloud Console → APIs & Services → Credentials (mirrored)

For mobile, the iOS reversed client ID and Android SHA-1 fingerprints
must be registered. Specifics recorded during Phase 1A and Phase 3E.
