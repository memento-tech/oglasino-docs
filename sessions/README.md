# Sessions

Archive of engineer session summaries. At the end of every engineer
session, Docs/QA copies that repo's `.agent/last-session.md` here,
renamed to `yyyy-mm-dd-<repo>-<slug>.md` per
[`../meta/conventions.md`](../meta/conventions.md) Part 5. The date
prefix sorts newest sessions to the top of the directory listing.
Files are archived verbatim — no reformatting.
